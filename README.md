# RemoteAccessTargetUnrealSample

빌드 테스트 환경
- 2024.08 ~
- Unreal 5.4.2
- Windows OS, Mac OS
- Sample Ver : 1.0

애플리케이션 실행 후 UI 노출 결과에 따른 기능
- Unreal의 Template 프로젝트인 FirstPerson 프로젝트를 사용
- Application 실행 시 Server에 연결히고 주기적으로 Server로 부터 데이터를 송신 받음
- Server에서 전달된 데이터 값에 따라 움직임 동작
